# VKontakte Cover Generator

Веб-приложение для генерации обложек ВКонтакте (разрешение 1920x768) с помощью Gemini AI.

## Запуск локально

**Требования:** Node.js

1. Установить зависимости: `npm install`
2. Создать файл `.env.local` и указать `GEMINI_API_KEY`
3. Запустить приложение: `npm run dev`

## Технологии

- React 19 + TypeScript
- Vite + Tailwind CSS v4
- Express.js
- Google Gemini AI API
- Framer Motion
